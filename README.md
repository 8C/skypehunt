# SkypeSearch
SkypeSearch is an async OSINT tool made to allow researchers to easily find information from a skype user, with just a username, phone number or email.

## What can I find?
- Skype ID
- Display Name
- Profile picture
- Location
- Date of Birth
- Gender
- Email address
- Creation Date
